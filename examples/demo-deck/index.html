How · 核心思路

把「看起来像什么」收进 :root

概念

每一种视觉属性——颜色、字体、圆角、阴影——都变成语义变量。

  • --text-1 / --text-2 / --text-3
  • --surface / --surface-2
  • --accent / --accent-2 / --accent-3
  • --radius / --shadow / --grad

示例

/* assets/themes/aurora.css */
:root {
  --bg: #06091c;
  --text-1: #e8f0ff;
  --accent: #5ef2c6;
  --accent-2: #7aa2ff;
  --accent-3: #c984ff;
  --radius: 20px;
}

——整个 aurora 主题就这么大。

how
关键是：base.css 只认变量名，不认具体色值。换主题 = 换一份变量。
